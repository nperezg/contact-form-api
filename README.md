# Contact Form API
Contact Form API developed with Symfony 4.

Requirements
------------
* PHP 7.1
* Composer

Run:

    composer install
    
**If you want to use the Symfony server:**    

Run server:

    php bin/console server:start
    
Use your favorite browser:

    http://127.0.0.1:8000/
    
Run tests:

    ./bin/phpunit
